# Nvim-config

Simple configuration for my needs.

This configuration is for neovim v0.11.0 (LuaJIT 2.1.1741730670)

## Instalation

1. Install Neovim. Follow the up-to-date [guide](https://github.com/neovim/neovim/blob/master/INSTALL.md)

Path to config file:

```
~./config/nvim/init.lua
```
